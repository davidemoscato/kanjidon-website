یہ مجھے بار بار وہی kanji کیوں دکھاتا ہے؟

یہ SRS system ہے جو تمہارے لیے کام کر رہا ہے! ہر kanji کی 3 independent skills ہوتی ہیں (Imi = meaning، Kun = Japanese پڑھنے کا طریقہ، On = Chinese-origin پڑھنے کا طریقہ)، ہر ایک کا اپنا review rhythm ہے۔ جب تم صحیح جواب دیتے ہو، interval بڑھتا ہے (1 day، 6، 15، 37، 93...)؛ اگر غلطی کرو، تو interval کم ہوتا ہے اور وہ skill زیادہ بار واپس آتی ہے۔

System ہمیشہ سب سے کمزور skill کو test کے لیے چنتا ہے، بالکل اس سے پہلے کہ تم واقعی بھول جاؤ۔ kana کے لیے صرف ایک skill ہے (pronunciation)، اس لیے وہ بہت آسان ہے۔

میرے پاس review کے لیے بہت زیادہ fragile cards ہیں، کیا کروں؟

1. Practice میں بڑے 'Review your fragile cards' circle کو tap کرو: یہ تمہیں سیدھا ان kanji تک لے جاتا ہے جن میں کم از کم ایک red ring ہو
2. چھوٹی مگر regular sessions کرو۔ روزانہ دس minutes، ہفتے میں ایک لمبے hour سے بہتر ہیں
3. جب تک backlog صاف نہ ہو، بہت زیادہ new packs نہ کھولو
4. theme کے حساب سے study کرنے کے لیے Decks استعمال کرو

Consistency اہم ہے: باقی کام SRS system کر دے گا!

cards پر وہ colored dots کیا ہیں؟

Dot kanji کی 3 rings (Imi، Kun، On) کا summary ہے: یہ ہمیشہ سب سے کمزور skill کا color دکھاتا ہے، کیونکہ جب تک ایک بھی skill fragile ہے، card کو review چاہیے۔

• ⚪ Grey = New (کوئز میں کبھی study نہیں کیا)
• 🔴 Red = Fragile (کم از کم ایک skill slip ہو رہی ہے، review کا وقت ہے)
• 🟡 Yellow = Solid (تم جانتے ہو، مگر اسے settle ہونے کے لیے مزید review چاہیے)
• 🟢 Green = Tamed (تم نے اسے conquer کر لیا!)

تمہارا goal ہر card کو Tamed بنانا اور وہیں رکھنا ہے۔ card tap کرو اور 3 rings detail میں دیکھنے کے لیے INFO press کرو!

ہر kanji میں 3 concentric Apple Watch-style rings ہوتی ہیں، ہر اس skill کے لیے ایک جسے تمہیں master کرنا ہے:

• Imi: kanji کا meaning
• Kun: Japanese پڑھنے کا طریقہ (kun'yomi)
• On: Chinese-origin پڑھنے کا طریقہ (on'yomi)

ہر ring تمہارے کوئز کے حساب سے خود بڑھتی اور گھٹتی ہے: ہو سکتا ہے تمہیں کسی kanji کا meaning perfectly آتا ہو مگر پڑھنے کا طریقہ نہیں، اور rings یہ بات فوراً دکھا دیتی ہیں۔ تم انہیں card کے INFO panel میں بڑا اور کوئز/اسباق میں چھوٹا دیکھو گے۔

اور جن kanji کی صرف ایک پڑھنے کا طریقہ ہو؟ وہاں تمہیں صرف 2 rings نظر آئیں گی۔ hiragana اور katakana کے لیے صرف ایک central ring ہوتی ہے (pronunciation)۔ یہ normal ہے!

rings وقت کے ساتھ خود empty کیوں ہوتی ہیں؟

کیونکہ human memory ایسے ہی کام کرتی ہے: جو چیزیں تم اچھی طرح جانتے ہو، وہ بھی دوبارہ نہ دیکھو تو fade ہو جاتی ہیں۔ rings اس decay کو ایمانداری سے دکھاتی ہیں، ہر دن review کے بغیر تھوڑا نیچے آتی ہیں۔

جب ring ایک certain threshold سے نیچے جاتی ہے، card Fragile بن جاتا ہے اور system اسے تمہارے کوئز میں صحیح وقت پر واپس لاتا ہے، بالکل اس سے پہلے کہ تم واقعی بھول جاؤ۔ skill جتنی solid تھی، اگلے review تک اتنا لمبا وقت ہوتا ہے۔

ایک important بات: جو skill red ہو گئی ہے وہ time کے ساتھ خود yellow واپس نہیں بنے گی۔ وہ red رہتی ہے جب تک تم review یا correct کوئز answer سے اسے fix نہ کرو۔ یہ design کا حصہ ہے: red action سے ختم ہوتا ہے، waiting سے نہیں۔

کیا kanji ہمیشہ کے لیے Tamed ہو جاتی ہے؟

System جانتا ہے کہ ہر skill کو کب واپس لانا ہے، لمبے ہوتے intervals کے ساتھ (1 day، 6، 15، 37، 93...)، بالکل اس سے پہلے کہ تم بھول جاؤ۔ جتنا زیادہ تم صحیح answer دیتے ہو، intervals اتنے لمبے ہوتے جاتے ہیں۔

Goal یہ نہیں کہ تمام rings ایک بار green کر کے ختم۔ Goal یہ ہے کہ انہیں وقت کے ساتھ full رکھو۔ یہ journey ہے، destination نہیں!

status filters کا کیا مطلب ہے؟
