한자부수: 2,000자가 200자가 됩니다

By David, Kanjidon Team · 2025년 12월 26일 · 12 min

2,136개의 한자를 배울 수 있습니다. 숫자만 읽어도 불안해집니다. 하지만 모든 것을 바꾸는 비밀이 있습니다. 그 2,136개의 문자는 모두 부수라고 불리는 동일한 214개의 기본 조각으로 만들어졌습니다. 부수를 배우고 단어보다 알파벳을 배우는 방법: 일단 조각을 알면 복잡한 한자는 신비한 디자인이 아닌 논리적 조합이 됩니다.

급진파란 무엇인가?

부수(부슈, PART首)는 한자의 기본 구성 요소입니다. 각 한자는 하나 이상의 부수로 구성됩니다. 그것들을 알파벳 글자처럼 생각하십시오. 그 자체로는 큰 의미가 없지만 결합하면 수천 개의 단어가 됩니다. 고전 시스템에는 214개(강희 부수)가 있지만 매일 연습하면 약 30-50개가 대부분의 한자를 포괄합니다.

각각의 부수에는 의미가 있으며, 종종 그것을 포함하는 한자의 의미나 범주를 암시합니다. 물기(氵)는 액체와 관련된 한자에 나타난다. 인간의 행위와 관련된 한자에는 인칭 과격(亻)이 등장한다. 이러한 패턴을 인식하는 것이 한자 공부의 진정한 초능력입니다.

급진주의자들이 모든 것을 바꾸는 이유

부수 없이 각 한자는 외울 수 있는 독특한 디자인입니다. 부수를 사용하면 각 한자는 이미 알고 있는 부분의 조합입니다. 다음은 세 가지 구체적인 예입니다.

예 1: 물 라디칼(水/氵)

라디칼 수(물)는 환원된 형태로 氵(왼쪽에 세 방울)로 나타납니다. 일단 인식하면 이 모든 한자는 직관적이 됩니다.

  • 海(바다) = 氵 water + 毎 모든 → 물 어디서나 = 바다
  • 泳(수영) = 氵 물 + 永 영원 → 영원히 물 속에서 움직이다 = 헤엄치다
  • 河(강) = 氵 물 + 可 가능 → 흐르는 물 = 강
  • 湖(호수) = 氵 물 + 胡 고대 → 고대, 고요한 물 = 호수
  • 波(파) = 氵 물 + 皮 피부 → 물 피부 = 파
  • 洗(세척) = 氵 물 + 先 먼저 → 물을 먼저 넣는다 = 씻다

한자 6개, 부수 1개. 의미 없는 그림 여섯 장을 외우는 대신 논리가 보입니다. 그리고 이것은 시작에 불과합니다.

사례 2: 나무의 뿌리(木)

부수적인 木(나무)는 자연, 나무, 식물과 관련된 많은 한자에 나타납니다.

  • 森(숲) = 木 + 木 + 木 → 세 그루의 나무 = 숲
  • rim(숲) = 木 + 木 → 두 그루의 나무 = 숲
  • 桜(벚꽃) = 木 + 女 + 女 → 여자의 나무(벚꽃은 아름다움과 연관됨)
  • 机(책상) = 木 + 几 → 나무 + 받침 = 책상
  • 板(판자/판자) = 木 + 反 → 편평한 나무 = 판자

예 3: 사람의 부수(人/亻)

옆자리의 부수 人은 亻가 됩니다. 인간의 행동과 조건과 관련된 많은 한자에 나타납니다.

  • 休(휴식) = 亻 사람 + 木나무 → 나무 아래서 쉬는 사람
  • 体(신체) = 亻 사람 + 本 유래 → 사람 유래 = 신체
  • 住(살다) = 亻 사람 + 主 주인 → 장소를 소유한 사람 = 살다
  • 작(do/create) = 亻 사람 + 乍 갑자기 → 만드는 사람 = ~을 하다
  • 何(무엇/무엇) = 亻 사람 + 可 가능 → 무엇을 묻는 사람과 가능 = 무엇?

Kanjidon은 각 부수에 대해 영어로 된 시각적 니모닉을 사용하여 올바른 순서로 부수를 가르칩니다. 이야기를 직접 만들 필요는 없습니다. 각 한자에 맞게 준비된 이야기를 찾을 수 있습니다.

가장 중요한 30가지 급진파

시작하는 데 214개 항목이 모두 필요하지는 않습니다. 이 30개의 부수는 N5-N3 레벨에서 접하게 될 한자의 약 80%를 차지합니다.

복잡한 한자를 기억하기 위해 부수를 사용하는 방법

방법은 간단합니다. 한자를 부수로 나눈 다음 이를 연결하는 이야기를 만들거나 찾는 것입니다. 다섯 가지 예를 자세히 살펴보겠습니다.

  • 休(휴식) = 亻 사람 + 木木 → 피곤한 사람이 나무 밑에 앉아 쉬는 모습을 상상해 보세요. 단순하고 시각적이며 잊을 수 없습니다.
  • 명(밝다) = 日해 + 月달 → 해와 달이 모두 보이면 빛이 두 배로 된다. 매우 밝습니다.
  • 好(기쁘다/좋다) = 女여자 + 子자녀 → 아이를 안고 있는 여자. 모자간의 유대감은 가장 아름다운 것입니다. 좋은.
  • verb(혀/언어) = 言 단어 + 五 five + 口 입 → 단어로 말하는 입이 5개입니다. 언어.
  • 体(몸) = ​​亻 사람 + 本 책/기원 → 사람의 근원, 그의 근본. 몸.

간지돈의 각 한자는 그 급진적인 구성요소와 이를 기억하기 위한 연상적 이야기를 보여줍니다. 아무것도 발명할 필요가 없습니다. 한자를 열고 이야기를 읽으면 나머지는 두뇌가 알아서 합니다. SRS를 사용하면 알고리즘은 사용자가 각 한자를 잊어버리려고 할 때 언제 복습하게 할지 결정합니다.

214명의 강시 급진파: 그들은 모두 복무하는가?

고전적인 강희주의 급진 체계는 총 214개입니다. 좋은 소식: 한자를 시작하기 전에 모든 내용을 배울 필요는 없습니다. 위 표의 30개는 N5에서 N3까지 만날 수 있는 대부분의 캐릭터를 포함합니다. 고급 한자에서 다른 사람을 만나면 배우십시오. 214개의 부수 중 다수는 드물거나 매우 적은 수의 한자로 나타납니다. 한꺼번에 모두 배우는 것은 시간 낭비입니다.

전략: 급진파를 연구하는 방법

  • 먼저 위의 표에서 30개의 기본기호를 배워보세요.
  • 각각의 의미와 형태를 기억하세요(물을 뜻하는 氵처럼 축소된 것도 포함)
  • 새로운 한자를 공부할 때는 항상 그것을 구성하는 부수를 식별하십시오
  • 부수를 한자의 의미와 연결하는 이야기를 만들거나 찾아보세요.
  • 몇 주 동안 따로 부수를 공부하지 마세요. 부수를 도구로 사용하여 바로 한자부터 시작하세요
  • 고급 한자에서 새로운 부수를 만나면 추가하세요.

급진파는 전제조건이 아니라 도구입니다. 첫 번째 한자부터 사용하면 비밀 무기가 될 것입니다. Kanjidon의 SRS 알고리즘은 레벨 N5에서 N1까지 모든 문자에 대한 니모닉 기록과 함께 실제 한자의 맥락에서 부수를 보여줍니다. 무료로 시작하여 한자가 더 이상 신비한 그림이 아니라 이미 알고 있는 부분의 논리적 조합일 때 얼마나 쉬운지 알아보세요.
