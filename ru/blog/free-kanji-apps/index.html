Бесплатные приложения для кандзи: Лучшие в 2026 году

Хочешь учить кандзи, но не хочешь тратить состояние на подписки? Хорошие новости: есть отличные бесплатные приложения (или почти бесплатные) для изучения японских иероглифов. Вот честное сравнение доступных вариантов.

На что обращать внимание в бесплатном приложении

Не все "бесплатные" приложения одинаковы. Некоторые полностью бесплатны, другие — freemium (бесплатные с ограничениями). Вот что оценивать:

  • Сколько контента реально бесплатно? — Только N5 или до N1?
  • Есть ли дневные лимиты? — Можно ли заниматься сколько хочешь?
  • Бесплатны ли важные функции? — SRS, тесты, офлайн?
  • Навязчивая реклама? — Реклама каждые 30 секунд?

Лучшие бесплатные приложения для кандзи

1. Anki (Десктоп + Android)

Стоимость: Бесплатно на десктопе и Android, $24.99 на iOS

Тип: Полностью настраиваемые SRS-карточки

Плюсы:

  • 100% бесплатно на десктопе и Android
  • Один из лучших SRS-алгоритмов
  • Колоды от сообщества
  • Полная настраиваемость

Минусы:

  • Устаревший, неинтуитивный интерфейс
  • Требует времени на настройку
  • Нет геймификации
  • Дорого на iOS ($24.99)

Лучше всего для: Технических пользователей, которые любят все настраивать.

2. Duolingo

Стоимость: Бесплатно с рекламой, Super $7.99/месяц

Тип: Общий языковой курс с кандзи

Плюсы:

  • Полностью бесплатно (с рекламой)
  • Красивый, геймифицированный интерфейс
  • Идеально для абсолютных новичков

Минусы:

  • Кандзи не в фокусе
  • Ограниченный охват (~N4)
  • SRS не оптимизирован для кандзи
  • Частая реклама в бесплатной версии

Лучше всего для: Тех, кто хочет начать с японского вообще.

3. Takoboto

Стоимость: Бесплатно (только Android)

Тип: Японский словарь с карточками

Плюсы:

  • Полностью бесплатно, без рекламы
  • Полный офлайн-словарь
  • Встроенные карточки

Минусы:

  • Только Android
  • Скорее словарь, чем приложение для учебы
  • Базовый SRS
  • Нет геймификации

Лучше всего для: Тех, кто ищет бесплатный словарь с карточками.

4. Kanji Study

Стоимость: Бесплатно (N5), $12.99 за все

Тип: База данных кандзи с тестами

Плюсы:

  • N5 полностью бесплатен
  • Полная, детальная база данных
  • Анимации порядка черт
  • Единоразовая покупка (без подписки)

Минусы:

  • Только Android
  • Бесплатен только N5
  • Скорее справочник, чем активная учеба
  • Базовые тесты

Лучше всего для: Тех, кто хочет детальный справочник по кандзи.

5. Kanjidon

Стоимость: Бесплатно с полными функциями, опциональный Premium

Тип: Геймифицированное приложение с SRS и PvP-битвами

Плюсы:

  • Все базовые функции бесплатны
  • Алгоритм SM-2 (научный SRS)
  • 8 типов тестов включены
  • Полный охват: кана → N1
  • Бесплатные PvP-битвы
  • Поддержка 21 языка
  • Работает офлайн
  • Без рекламы

Минусы:

  • Некоторые продвинутые функции требуют Premium
  • Относительно новое приложение

Лучше всего для: Тех, кто хочет полноценное современное приложение без оплаты.
