5 лучших приложений для изучения кандзи в 2026 году

Ты решил изучать кандзи. Отличный выбор—и плохое время для твоего рассудка. С более чем 2000 иероглифов, которые нужно запомнить только для чтения газеты, тебе понадобится хороший цифровой помощник. Но какое приложение выбрать из десятков доступных? Вот честное сравнение, без прикрас.

Что искать в приложении для кандзи

  • Интервальное повторение (SRS) — Единственный научно доказанный метод долгосрочного запоминания
  • Разнообразие упражнений — Чтение, значение, аудирование, письмо
  • Охват JLPT — От N5 до N1, без смены приложения
  • Встроенная мотивация — Серии, цели, награды
  • Доступная цена — Потому что изучение японского и так требует много усилий

1. Duolingo

Вердикт: Отлично для начала, недостаточно для кандзи.

Плюсы: Привлекательный интерфейс, бесплатно, подходит для начинающих.

Минусы: Кандзи—не фокус, нет отдельной системы SRS, ограниченный охват (до ~N4).

2. Anki

Вердикт: Невероятно мощный, но требует диплома по настройке.

Плюсы: Один из лучших алгоритмов SRS, полностью настраиваемый, бесплатно на десктопе и Android.

Минусы: Устаревший интерфейс, крутая кривая обучения, iOS-приложение стоит $25, нулевая геймификация.

3. WaniKani

Вердикт: Отличный метод, премиум-цена, только на английском.

Плюсы: Система радикалов, креативные мнемоники, структурированный путь.

Минусы: ~$9/месяц, только английский, жёсткий темп, нет официального приложения.

4. Kanji Study

Вердикт: Отличный словарь, менее эффективен как инструмент активного обучения.

Плюсы: Полная база данных, анимации штрихов, работает офлайн, единоразовая покупка.

Минусы: Больше словарь, базовые тесты, нет геймификации, только Android.

5. Kanjidon

Вердикт: Современный подход—серьёзная геймификация встречает научный SRS.

Плюсы

  • 8 типов тестов—значение, чтение on/kun, аудирование, письмо
  • Алгоритм SM-2—научно проверенная система SRS
  • Коллекционируй кандзи как карты
  • PvP-баталии в реальном времени—ранги от E до S
  • 21 язык—учись на родном языке
  • Полный охват—от каны до JLPT N1
  • Бесплатно с полными базовыми функциями
  • Нативные приложения iOS и Android, работает офлайн

Минусы

  • Не использует повествовательные мнемоники
  • PvP-баталии требуют интернета
  • Относительно новое приложение
