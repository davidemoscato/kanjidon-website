Кандзи-радикалы: 2000 символов становятся 200

By David, Kanjidon Team · 26 дек. 2025 г. · 12 min

2136 кандзи, которые нужно выучить. Просто читая номер, ты начинаешь беспокоиться. Но есть секрет, который меняет все: все эти 2136 символов состоят из одних и тех же 214 основных частей, называемых радикалами. Изучение радикалов и как учить алфавит раньше слов: как только ты узнаешь части, сложные кандзи станут логическими комбинациями, а не загадочными узорами.

Что такое радикалы?

Радикалы (бусю, 部首) являются основными компонентами кандзи. Каждый кандзи состоит из одного или нескольких радикалов. Думай о них как о буквах алфавита: сами по себе они мало что значат, но в сочетании они составляют тысячи слов. В классической системе их 214 (радикалы Канси), но в повседневной практике около 30-50 охватывают подавляющее большинство кандзи, с которыми ты столкнёшься.

Каждый радикал имеет значение и часто предполагает значение или категорию кандзи, который его содержит. Водный радикал (氵) появляется в кандзи, относящихся к жидкостям. Радикальный человек (亻) появляется в кандзи, связанных с человеческими действиями. Распознавание этих закономерностей, настоящая суперсила в изучении кандзи.

Почему радикалы меняют все

Если не считать радикалов, каждый кандзи представляет собой уникальный дизайн для запоминания. В радикалах каждый кандзи представляет собой комбинацию уже известных тебе частей. Вот три конкретных примера.

Пример 1: Водный радикал (水/氵)

Радикал 水 (вода) появляется в сокращенной форме как 氵 (три капли слева). Как только ты это узнаешь, все эти кандзи станут интуитивно понятными:

  • 海 (море) = 氵 вода + 毎 каждый → вода повсюду = море
  • 泳 (плавать) = 氵 вода + 永 вечный → двигаться в воде вечно = плавать
  • 河 (река) = 氵 вода + 可 возможно → текущая вода = река
  • 湖 (озеро) = 氵 вода + 胡 древний → древняя, стоячая вода = озеро
  • 波 (волна) = 氵 вода + 皮 кожа → кожа воды = волна
  • 洗 (мыть) = 氵 вода + 先 сначала → сначала положить воду = мыть

Шесть кандзи, один радикал. Вместо запоминания шести бессмысленных рисунков ты видишь логику. И это только начало.

Пример 2: Корень дерева (木)

Радикальный 木 (дерево) встречается во многих кандзи, связанных с природой, деревом и растениями:

  • 森 (лес) = 木 + 木 + 木 → три дерева = лес
  • 林 (лес) = 木 + 木 → два дерева = лес
  • 桜 (вишневое дерево) = 木 + 女 + 女 → женское дерево (цветение вишни ассоциировалось с красотой)
  • 机 (стол) = 木 + 几 → дерево + опора = стол
  • 板 (доска/доска) = 木 + 反 → плоское дерево = доска

Пример 3: Радикал личности (人/亻)

Радикал 人 (человек) в латеральной позиции становится 亻. Оно появляется во многих кандзи, связанных с человеческими действиями и условиями:

  • 休 (отдых) = 亻 человек + 木 дерево → человек, отдыхающий под деревом
  • 体 (тело) = 亻 человек + 本 происхождение → происхождение человека = тело
  • 住 (жить) = 亻 человек + 主 владелец → человек, которому принадлежит место = жить
  • 作 (делать/создать) = 亻 человек + 乍 внезапно → человек, который создает = делать
  • 何 (что/что) = 亻 человек + 可 возможный → человек, который спрашивает, что и возможно = что?

Как использовать радикалы для запоминания сложных кандзи

Метод прост: разбей кандзи на радикалы, затем создай (или найди) историю, которая их связывает. Вот пять примеров в деталях.

  • 休 (отдых) = 亻 человек + 木 дерево → Представь себе уставшего человека, сидящего под деревом, чтобы отдохнуть. Просто, наглядно, невозможно забыть.
  • 明 (яркий) = 日 солнце + 月 луна → Когда видно и солнце, и луну, свет двойной. Очень яркий.
  • 好 (приятный/хороший) = 女 женщина + 子 ребенок → Женщина с ребенком. Связь матери и сына - это самое прекрасное. Хороший.
  • 語 (язык/язык) = 言 слово + 五 пять + 口 рот → Пять ртов, говорящих словами. Язык.
  • 体 (тело) = 亻 человек + 本 книга/происхождение → Происхождение человека, его основа. Тело.

Каждый кандзи в кандзидоне показывает свои радикальные компоненты и мнемоническую историю, позволяющую его запомнить. Тебе не нужно ничего придумывать: открой кандзи, прочитай историю, а твой мозг сделает все остальное. С помощью SRS алгоритм решает, когда тебе следует просмотреть каждый кандзи, как раз тогда, когда ты собираетесь его забыть.

214 радикалов Канси: все ли они служат?

Всего в классической радикальной системе Канси их 214. Хорошая новость: тебе не обязательно учить их все, прежде чем начать изучать кандзи. 30 в таблице выше охватывают подавляющее большинство персонажей, с которыми ты встретишься, от N5 до N3. Изучай других, когда сталкиваетесь с ними, используя более сложные кандзи. Многие из 214 радикалов редки или встречаются в очень небольшом количестве кандзи, изучение их всех одновременно было бы пустой тратой времени.

Стратегия: как изучать радикалы

  • Сначала изучи 30 фундаментальных радикалов из таблицы выше.
  • Для каждого из них запомни значение и форму (даже сокращенную, например 氵 для воды).
  • Изучая новый кандзи, всегда идентифицируй радикалы, входящие в его состав.
  • Создай или найди историю, которая связывает радикалы со значением кандзи.
  • Не изучай радикалы изолированно неделями: начни с кандзи прямо сейчас, используя радикалы как инструмент.
  • Добавляй новые радикалы по мере их появления в более сложных кандзи.

Радикалы не являются обязательным условием: они являются инструментом. Используй их из первого кандзи, и они станут твоим секретным оружием. Алгоритм SRS Kanjidon показывает тебе радикалы в контексте реальных кандзи с мнемоническими историями для каждого персонажа от уровня N5 до N1. Начни бесплатно и узнай, насколько это проще, когда кандзи больше не представляют собой загадочные рисунки, а представляют собой уже знакомые тебе логические комбинации частей.
