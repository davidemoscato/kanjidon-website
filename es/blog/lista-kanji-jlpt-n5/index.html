Lista Kanji JLPT N5: 100 Kanji Esenciales

El JLPT N5 es el primer paso en tu viaje hacia el dominio del japones. Requiere conocer aproximadamente 100 kanji. Aqui tienes la lista completa con significados y lecturas, organizada por categorias.

Que es el JLPT N5?

El Japanese Language Proficiency Test (JLPT) es el examen oficial de japones reconocido mundialmente. El nivel N5 es el nivel basico, disenado para quienes:

  • Pueden leer hiragana y katakana
  • Conocen las frases y expresiones mas comunes
  • Pueden entender conversaciones sencillas
