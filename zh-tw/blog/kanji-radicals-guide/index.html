漢字部首：新手完整指南

學習漢字看起來像是不可能的任務：成千上萬個看似隨機的字符。但有一個秘密是日本小學生就學到的：部首。它們就像組成所有漢字的「樂高積木」。

什麼是部首？

部首（部首，bushu）是漢字的基本組成部分。每個漢字都由一個或多個部首組成。傳統上共有214個部首，繼承自中國的分類系統。

把部首想成漢字的字母表：就像26個字母可以組成所有英文單詞一樣，214個部首可以組成所有漢字。

實際範例

漢字「休」（休息）由以下組成：

  • 亻（「人」部首）在左邊
  • 木（「樹」部首）在右邊

視覺意義：一個人（亻）靠著一棵樹（木）= 休息。部首不僅組成漢字——它們通常還暗示意義！

為什麼部首如此重要

1. 讓漢字更容易記憶

與其死記2,000個隨機的圖形，不如學習214個組件及其組合。這就像記住2,000個隨機密碼和理解其背後邏輯的差別。

2. 幫助猜測意義

許多部首表示語義類別：

  • 氵（水）→ 海（海洋）、河（河流）、泳（游泳）
  • 火（火）→ 焼（燒）、煙（煙）、炎（火焰）
  • 言（言語）→ 話（說話）、語（語言）、読（閱讀）
  • 心（心）→ 思（思考）、感（感覺）、愛（愛）

3. 幫助猜測讀音

某些組件（稱為「聲符」）可以提示音讀：

  • 工（kō）→ 紅（kō）、功（kō）、空（kū）
  • 青（sei）→ 清（sei）、精（sei）、晴（sei）

4. 能夠查字典

傳統日語字典按部首排列漢字。了解部首可以讓你查找不認識的漢字。

部首的位置

部首可以出現在漢字的不同位置：

  • 偏（へん） — 左側：休（亻在左）
  • 旁（つくり） — 右側：新（斤在右）
  • 冠（かんむり） — 上方：雪（雨在上）
  • 脚（あし） — 下方：思（心在下）
  • 垂（たれ） — 左上：病（疒）
  • 繞（にょう） — 左下：道（辶）
  • 構（かまえ） — 包圍：国（囗）

如何有效學習部首

常見錯誤：孤立學習部首

很多學生試圖在開始學漢字之前先記住所有214個部首。這行不通。部首只有在它們所組成的漢字的語境中才有意義。

更好的方法：隨漢字一起學部首

學習新漢字時：

  1. 識別組成部分（部首）
  2. 記下每個部首的位置
  3. 尋找與意義的關聯
  4. 與你已經認識的類似漢字比較
