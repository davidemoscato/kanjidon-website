日語助詞終極指南：徹底搞懂 に、で、を、は、が

By David, Kanjidon Team · 2025年12月15日 · 10 分鐘

助詞是每個日語學習者的噩夢。用 に 還是 で？是 は 還是 が？什麼時候該用 を？如果你花了數小時研究它們的區別卻依然頭大，你並不孤單。問題在於大多數解釋太理論化了。這裡有助詞的實際用法，以及你可以立即套用的案例。

什麼是助詞？

助詞（助詞，joshi）是位於詞語之後的短小詞彙，用來指示該詞的語法功能。在漢語中，我們通常靠語序或虛詞；而在日語中，助詞承擔了所有髒活累活。用錯一個助詞，句子的意思就可能天差地別。

は (wa) - 主題標記

は 用來標記主題，意思是「我們要談論的內容」。它不一定是語法主語（那是 が）。它就像是在句首說：「關於 X 嘛……」

  • 私は学生です (Watashi wa gakusei desu) - 關於我，我是學生
  • 今日は暑いです (Kyō wa atsui desu) - 關於今天，天氣很熱
  • この本は面白い (Kono hon wa omoshiroi) - 關於這本書，很有趣

が (ga) - 主語標記

が 用來標記執行動作或擁有屬性的主語。當主語是新信息時、回答「誰？」或「什麼？」時，或者配合某些特定動詞（如 わかる、ある、いる、好き、欲しい）時使用它。

  • 谁が来ましたか？田中さんが来ました (谁來了？田中先生來了)
  • 日本語がわかります (懂日語 - 字面意：日語是可理解的)
  • 猫が好きです (喜歡貓 - 字面意：貓是惹人喜愛的)

は vs が：實操準則

簡單規則：は 用於舊信息/已知信息， が 用於新信息。「那隻貓吃了魚」 → 猫は魚を食べた（我們已經知道那隻貓了）。「有一隻貓吃了魚」 → 猫が魚を食べた（第一次提到這隻貓）。

を (wo/o) - 直接賓語標記

を 用來標記直接賓語，即動作的直接承受者。如果你在動詞後面可以問「什麼？」，那麼那個詞前面就要加 を。

  • りんごを食べる (吃蘋果) - 吃什麼？蘋果
  • 本を読む (讀書) - 讀什麼？書
  • 日本語を勉強する (學日語) - 學什麼？日語

注意：を 也可以用來表示穿越某地：公園を歩く (在公園散步)，道を渡る (過馬路)。

に (ni) - 目的地、時間、存在

に 是功能最豐富的助詞。它的主要用途有三個：目的地（去哪兒）、具體時間（什麼時候）以及存在的地點（某物在哪裡）。

表示目的地

  • 学校に行く (去學校) - 移動的目的地
  • 日本に来た (來到日本)
  • 家に帰る (回家)

表示具體時間

  • 7時に起きる (7點起床)
  • 月曜日に会いましょう (週一見吧)
  • 2024年に日本に行った (2024年去了日本)

表示存在

  • 東京に住んでいる (住在東京)
  • 机の上に本がある (桌子上有本書)
  • 公園に猫がいる (公園裡有隻貓)

で (de) - 動作地點、手段、原因

で 用來表示動作發生的地點、你所使用的工具、或者事情發生的原因。它和 に 的區別至關重要：に 是靜態的（某物「在」哪）， で 是動態的（某事「在」哪發生）。

表示動作發生地點

  • 図書館で勉強する (在圖書館學習) - 動作在那兒發生
  • レストランで食べる (在餐廳吃飯)
  • 公園で遊ぶ (在公園玩)

表示手段/工具

  • 箸で食べる (用筷子吃)
  • 電車で行く (乘電車去)
  • 日本語で話す (用日語說)

表示原因/理由

  • 病気で休んだ (因病請假)
  • 雨で試合が中止になった (比賽因雨取消)

に vs で：深度對比

場景にで
我住的地方東京に住む ✓×
我學習的地方×図書館で勉強する ✓
某物在那兒机の上にある ✓×
在那兒發生了某事×ここで事故があった ✓

記憶秘訣：に = 到達點/存在點（靜態）。で = 動作舞台（動態）。如果動作在那兒「發生」，用 で；如果某物「在/去」那兒，用 に。

其他必備助詞

へ (e) - 方向

類似於表示方向的 に，但更強調「朝著」某個方向。日本へ行く 強調方向，日本に行く 強調目的地。實際操作中兩者常互換。

と (to) - 和、伴隨

  • 友達と行く (和朋友一起去)
  • りんごとバナナ (蘋果和香蕉)
  • 彼と话した (和他說了話)

から (kara) 和 まで (made) - 從……到……

  • 9時から5時まで働く (從 9 點工作到 5 點)
  • 東京から大阪まで (從東京到大阪)
  • 子供から大人まで (從小孩到大人)

の (no) - 領屬和連接

  • 私の本 (我的書)
  • 日本の文化 (日本文化)
  • 友達の友達 (朋友的朋友)

常見錯誤避雷

  • ❌ 学校で行く → ✓ 学校に行く (目的地 = に)
  • ❌ 図書館に勉強する → ✓ 図書館で勉強する (動作發生地 = で)
  • ❌ 日本語を話す人は... → ✓ 日本語が话せる人は... (能力相關 = が)
  • ❌ 私が田中です → ✓ 私は田中です (介紹主題 = は，除非回答「誰是？」)

如何練習

助詞不是靠背出來的，而是靠用出來的。每次讀或聽日語時，問問自己：為什麼那兒用了這個助詞？通過 Kanjidon，你可以在真實詞彙和句子的語境中學習漢字，潛移默化地接觸助詞的實戰用法。

總結

助詞看起來複雜，是因為我們在中文裡尋找不存在的完全對應項。別再試圖直譯了，去感受它們的功能：に 指向，で 承載，を 接受，は 呈現，が 鎖定。勤加練習，這終將成為你的直覺。
