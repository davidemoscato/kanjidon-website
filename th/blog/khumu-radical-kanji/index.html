อนุมูลคันจิ: 2,000 ตัวอักษรกลายเป็น 200

By David, Kanjidon Team · 26 ธ.ค. 2568 · 12 min

2,136 ตัวอักษรคันจิที่จะเรียนรู้ แค่อ่านตัวเลขก็กังวลแล้ว แต่มีความลับที่เปลี่ยนแปลงทุกสิ่ง: ตัวละคร 2,136 ตัวนั้นถูกสร้างขึ้นจากชิ้นส่วนพื้นฐาน 214 ชิ้นเดียวกันที่เรียกว่าอนุมูล การเรียนรู้รากศัพท์และวิธีการเรียนรู้ตัวอักษรก่อนคำศัพท์: เมื่อคุณรู้ส่วนต่างๆ แล้ว คันจิที่ซับซ้อนจะกลายเป็นการผสมผสานที่สมเหตุสมผล ไม่ใช่การออกแบบที่ลึกลับ

อนุมูลคืออะไร?

Radicals (bushu, 部首) เป็นองค์ประกอบพื้นฐานของตัวอักษรคันจิ คันจิแต่ละตัวประกอบด้วยอนุมูลตั้งแต่หนึ่งตัวขึ้นไป คิดว่าพวกมันเหมือนกับตัวอักษรในตัวอักษร: ด้วยตัวมันเองพวกมันไม่ได้มีความหมายอะไรมาก แต่เมื่อรวมกันพวกมันก็สร้างคำศัพท์ได้หลายพันคำ ระบบคลาสสิกมี 214 (รากของ Kangxi) แต่ในทางปฏิบัติในชีวิตประจำวันประมาณ 30-50 ครอบคลุมตัวอักษรคันจิส่วนใหญ่ที่คุณจะพบ

รากศัพท์แต่ละตัวมีความหมาย และมักจะแนะนำความหมายหรือหมวดหมู่ของคันจิที่มีอยู่ อนุมูลน้ำ (氵) ปรากฏในตัวอักษรคันจิที่เกี่ยวข้องกับของเหลว บุคคลหัวรุนแรง (亻) ปรากฏในตัวอักษรคันจิที่เกี่ยวข้องกับการกระทำของมนุษย์ การตระหนักถึงรูปแบบเหล่านี้เป็นพลังพิเศษที่แท้จริงในการเรียนคันจิ

เหตุใด Radicals จึงเปลี่ยนแปลงทุกสิ่ง

คันจิแต่ละตัวได้รับการออกแบบให้จดจำอย่างมีเอกลักษณ์ ด้วยเครื่องหมายราก ตัวอักษรคันจิแต่ละตัวคือการรวมกันของชิ้นส่วนที่คุณรู้จักอยู่แล้ว นี่เป็นตัวอย่างที่ชัดเจนสามตัวอย่าง

ตัวอย่างที่ 1: หัวรุนแรงของน้ำ (水/氵)

รากศัพท์ 水 (น้ำ) ปรากฏในรูปแบบย่อเป็น 氵 (สามหยดทางด้านซ้าย) เมื่อคุณจำได้แล้ว คันจิทั้งหมดเหล่านี้จะกลายมาเป็นสัญชาตญาณ:

  • 海 (ทะเล) = 氵 น้ำ + 毎 ทุก → น้ำทุกที่ = ทะเล
  • 泳 (ว่ายน้ำ) = 氵 น้ำ + 永 ชั่วนิรันดร์ → เคลื่อนตัวในน้ำตลอดไป = ว่ายน้ำ
  • 河 (แม่น้ำ) = 氵 น้ำ + 可 เป็นไปได้ → น้ำไหล = แม่น้ำ
  • 湖 (ทะเลสาบ) = 氵 น้ำ + 胡 โบราณ → น้ำนิ่ง = ทะเลสาบ
  • 波 (คลื่น) = 氵 น้ำ + 皮 ผิว → ผิวน้ำ = คลื่น
  • 洗 (ล้าง) = 氵 น้ำ + 先 ก่อน → ใส่น้ำก่อน = ล้าง

คันจิหกตัว หนึ่งราก แทนที่จะท่องจำภาพวาดที่ไร้ความหมายหกภาพ คุณจะมองเห็นตรรกะ และนี่เป็นเพียงจุดเริ่มต้น

ตัวอย่างที่ 2: รากของต้นไม้ (木)

รากศัพท์ 木 (ต้นไม้) ปรากฏในตัวอักษรคันจิหลายตัวที่เกี่ยวข้องกับธรรมชาติ ไม้ และพืช:

  • 森 (ป่า) = 木 + 木 + 木 → ต้นไม้สามต้น = ป่า
  • 林 (ป่า) = 木 + 木 → ต้นไม้สองต้น = ป่า
  • 桜 (ต้นซากุระ) = 木 + 女 + 女 → ต้นของผู้หญิง (ดอกซากุระเกี่ยวข้องกับความงาม)
  • 机 (โต๊ะ) = 木 + 几 → ไม้ + ตัวรองรับ = โต๊ะ
  • 板 (ไม้กระดาน/กระดาน) = 木 + 反 → ไม้เรียบ = ไม้กระดาน

ตัวอย่างที่ 3: หัวรุนแรงของบุคคล (人/亻)

人 หัวรุนแรง (บุคคล) ในตำแหน่งด้านข้างกลายเป็น 亻ปรากฏในตัวอักษรคันจิหลายตัวที่เกี่ยวข้องกับการกระทำและเงื่อนไขของมนุษย์:

  • 休 (พักผ่อน) = 亻 คน + 木 ต้นไม้ → คนพักผ่อนใต้ต้นไม้
  • 体 (ร่างกาย) = 亻 คน + 本 ต้นกำเนิด → ต้นกำเนิดของบุคคล = ร่างกาย
  • 住 (อยู่) = 亻 คน + 主 เจ้าของ → คนที่เป็นเจ้าของ = อยู่
  • 作 (ทำ/สร้าง) = 亻 คน + 乍 จู่ๆ → คนสร้าง = ทำ
  • 何 (อะไร/อะไร) = 亻 คน + 可 เป็นไปได้ → คนที่ถามอะไร และเป็นไปได้ = อะไร?

คันจิด้งจะสอนคุณเกี่ยวกับอนุมูลตามลำดับที่ถูกต้อง โดยมีภาพช่วยจำเป็นภาษาอังกฤษสำหรับแต่ละรายการ คุณไม่จำเป็นต้องประดิษฐ์เรื่องราวขึ้นมาเอง เพราะคุณสามารถค้นหาเรื่องราวเหล่านั้นให้พร้อมสำหรับคันจิแต่ละตัวได้

30 อนุมูลที่สำคัญที่สุด

คุณไม่จำเป็นต้องมีทั้งหมด 214 เพื่อเริ่มต้น อนุมูล 30 ตัวนี้ครอบคลุมประมาณ 80% ของตัวอักษรคันจิที่คุณจะพบในระดับ N5-N3

วิธีใช้ Radicals เพื่อจำตัวอักษรคันจิที่ซับซ้อน

วิธีการนั้นง่ายมาก: แบ่งคันจิออกเป็นราก จากนั้นสร้าง (หรือค้นหา) เรื่องราวที่เชื่อมโยงพวกมัน ต่อไปนี้เป็นตัวอย่างโดยละเอียด 5 ตัวอย่าง

  • 休 (พักผ่อน) = 亻 คน + 木 ต้นไม้ → ลองนึกภาพคนเหนื่อยๆ นั่งอยู่ใต้ต้นไม้เพื่อพักผ่อน เรียบง่าย มองเห็นได้ ไม่อาจลืมเลือน
  • 明 (สว่าง) = 日ดวงอาทิตย์ + 月 ดวงจันทร์ → เมื่อมองเห็นดวงอาทิตย์และดวงจันทร์ แสงจะเพิ่มเป็นสองเท่า สดใสมาก.
  • 好 (น่าพอใจ/ดี) = 女 ผู้หญิง + 子 ลูก → ผู้หญิงกับลูก ความผูกพันแม่ลูกคือสิ่งที่สวยงามที่สุด ดี.
  • 語 (ลิ้น/ภาษา) = 言 คำ + 五 ห้า + 口 ปาก → ห้าปากที่พูดด้วยคำพูด ภาษา.
  • 体 (ร่างกาย) = 亻 คน + 本 หนังสือ/ต้นกำเนิด → ต้นกำเนิดของบุคคล พื้นฐานของเขา ร่างกาย.

คันจิแต่ละตัวในคันจิด้งจะแสดงองค์ประกอบที่ต่างไปจากเดิมอย่างสิ้นเชิงและเรื่องราวช่วยในการจำ คุณไม่จำเป็นต้องประดิษฐ์อะไรเลย แค่เปิดตัวอักษรคันจิ อ่านเรื่องราว แล้วสมองของคุณก็จะจัดการส่วนที่เหลือ ด้วย SRS อัลกอริธึมจะตัดสินใจว่าเมื่อใดที่จะทำให้คุณทบทวนตัวอักษรคันจิแต่ละตัว และเมื่อคุณกำลังจะลืมมัน

214 อนุมูลคังซี: พวกเขาทั้งหมดทำหน้าที่หรือไม่?

ระบบรากของคังซีแบบคลาสสิกมีทั้งหมด 214 ระบบ ข่าวดี: คุณไม่จำเป็นต้องเรียนรู้ทั้งหมดก่อนที่จะเริ่มเรียนคันจิ 30 ตัวในตารางด้านบนครอบคลุมอักขระส่วนใหญ่ที่คุณจะพบตั้งแต่ N5 ถึง N3 เรียนรู้ผู้อื่นเมื่อคุณเผชิญหน้าพวกเขาด้วยตัวอักษรคันจิขั้นสูง ตัวอักษรคันจิทั้ง 214 ตัวนั้นหายากหรือมีตัวคันจิน้อยมาก การเรียนรู้พวกมันทั้งหมดในคราวเดียวจะเสียเวลา

กลยุทธ์: วิธีการศึกษาหัวรุนแรง

  • ขั้นแรกให้เรียนรู้อนุมูลพื้นฐาน 30 ตัวจากตารางด้านบน
  • แต่ละตัวให้จำความหมายและรูป (แม้ตัวย่อ เช่น 氵 แปลว่า น้ำ)
  • เมื่อศึกษาคันจิใหม่ ให้ระบุรากที่ประกอบขึ้นเสมอ
  • สร้างหรือค้นหาเรื่องราวที่เชื่อมโยงอนุมูลกับความหมายของคันจิ
  • อย่าศึกษาอนุมูลแยกกันเป็นเวลาหลายสัปดาห์: เริ่มด้วยตัวคันจิทันทีโดยใช้อนุมูลเป็นเครื่องมือ
  • เพิ่มรากใหม่เมื่อคุณพบพวกมันด้วยตัวอักษรคันจิขั้นสูง

Radicals ไม่ใช่สิ่งที่จำเป็นต้องมี แต่เป็นเครื่องมือ ใช้พวกมันตั้งแต่คันจิตัวแรกแล้วพวกมันจะกลายเป็นอาวุธลับของคุณ อัลกอริธึม SRS ของคันจิดอนจะแสดงรากศัพท์ในบริบทของคันจิจริง พร้อมประวัติช่วยในการจำสำหรับตัวละครทุกตัวตั้งแต่ระดับ N5 ถึง N1 เริ่มต้นฟรีและค้นพบว่ามันจะง่ายแค่ไหนเมื่อตัวอักษรคันจิไม่ใช่ภาพวาดลึกลับอีกต่อไป แต่เป็นการผสมผสานตรรกะของชิ้นส่วนที่คุณรู้จักอยู่แล้ว
